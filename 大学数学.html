大学数学复习资料

第一章 函数、极限与连续

第一节 函数

一、函数的概念及特性

  • 函数的定义：设 ( x ) 和 ( y ) 是两个变量，( D ) 是一个给定的数集。如果对于每个数 ( x \in D )，变量 ( y ) 按照一定法则总有（唯一）确定的数值和它对应，则称 ( y ) 是 ( x ) 的函数，记作 ( y = f(x) )，数集 ( D ) 叫这个函数的定义域，( x ) 叫做自变量，( y ) 叫做因变量。( f(a) ) 表示函数 ( f(x) ) 在 ( x = a ) 时的函数值，( f(D) = { y | y = f(x), x \in D } ) 称为函数的值域，定义域 ( D ) 和对应法则 ( f ) 是函数关系的两要素。
  • 函数的表示方法：
    • 公式法：如 ( y = 3x^2 + 2x - 4 )
    • 表格法：如月份与销售量的对应关系
    • 图象法：如函数的图像
  • 函数的几种特性：
    • 有界性：函数的值域有上界或下界
    • 单调性：函数在定义域内单调递增或单调递减
    • 奇偶性：奇函数满足 ( f(-x) = -f(x) )，偶函数满足 ( f(-x) = f(x) )
    • 周期性：函数满足 ( f(x + T) = f(x) )，其中 ( T ) 为周期

二、基本初等函数

  • 幂函数：( y = x^\mu )，如 ( y = x^2 )
  • 指数函数：( y = a^x )（( a > 0, a \neq 1 )），如 ( y = 2^x )
  • 对数函数：( y = \log_a(x) )（( a > 0, a \neq 1 )），( y = \ln(x) )，如 ( y = \log_2(x) )
  • 三角函数：( y = \sin(x) )，( y = \cos(x) )，( y = \tan(x) )，( y = \cot(x) )，( y = \sec(x) )，( y = \csc(x) )
  • 反三角函数：( y = \arcsin(x) )，( y = \arccos(x) )，( y = \arctan(x) )，( y = \text{arccot}(x) )

三、分段函数

  • 定义：在自变量的不同变化范围中，对应法则用不同式子来表示的函数，称为分段函数。例如： [ f(x) = \begin{cases} x^2 & \text{if } 0 \leq x \leq 1 \ x + 1 & \text{if } x > 1 \end{cases} ]
  • 求分段函数的函数值：要根据自变量的取值范围，确定相应的函数表达式。例如，对于上述函数，( f(2) = 2 + 1 = 3 )。

四、函数的复合

  • 定义：由常数和基本初等函数出发经过有限次四则运算和有限次的函数复合步骤所构成并可用一个式子表示的函数称为初等函数。例如： [ y = \sin(2x) ] 可以看作是 ( y = \sin(u) ) 和 ( u = 2x ) 的复合函数。
  • 复合函数的分解：将复合函数分解为简单函数。例如： [ y = \sin^2(\ln(x)) ] 可以分解为 ( y = u^2 )，( u = \sin(v) )，( v = \ln(x) )。

第二节 极限

一、极限的概念

  • 数列极限的定义：设有数列 ({a_n}) 和常数 (A)，如果对任意给定的正数 (\varepsilon)，总存在一个正整数 (N)，使得对于 (n > N) 时的一切 (a_n)，总有 (|a_n - A| < \varepsilon) 成立，则称常数 (A) 是数列 ({a_n}) 的极限，或称数列 ({a_n}) 收敛于 (A)，记作 (\lim_{n \to \infty} a_n = A)。如果数列没有极限，就称数列是发散的。
  • 函数的极限：类似于数列极限，如果在自变量的某个变化过程中，对应的函数值可以无限接近于某个确定的常数，那么这个确定的常数就叫做函数在该变化过程中的极限。例如： [ \lim_{x \to \infty} \frac{1}{x} = 0 ]

二、极限的运算法则

  • 四则运算法则：
    • (\lim_{x \to a} [f(x) \pm g(x)] = \lim_{x \to a} f(x) \pm \lim_{x \to a} g(x))
    • (\lim_{x \to a} [f(x) \cdot g(x)] = \lim_{x \to a} f(x) \cdot \lim_{x \to a} g(x))
    • (\lim_{x \to a} \frac{f(x)}{g(x)} = \frac{\lim_{x \to a} f(x)}{\lim_{x \to a} g(x)})（如果 (\lim_{x \to a} g(x) \neq 0)）
  • 复合函数的极限运算法则：如果函数 (y = f(u)) 在 (u = g(a)) 处连续，且 (\lim_{x \to a} g(x) = b)，则 (\lim_{x \to a} f(g(x)) = f(b))。

三、无穷小与无穷大

  • 无穷小：如果函数 (f(x)) 在某个极限过程中的极限为零，那么就称 (f(x)) 是此极限过程的无穷小（量）。例如，当 (x \to 0) 时，(x) 是无穷小。
  • 无穷大：如果函数 (f(x)) 在某个极限过程中，对应的函数值的绝对值可以无限增大，那么就称 (f(x)) 是此极限过程的无穷大（量）。例如，当 (x \to 0) 时，(\frac{1}{x}) 是无穷大。
  • 无穷小与无穷大的关系：在同一极限过程中，无穷小与无穷大之间是通过取倒数互相转化。例如，当 (x \to 0) 时，(x) 是无穷小，(\frac{1}{x}) 是无穷大。

四、极限存在准则

  • 夹逼准则：如果 (g(x) \leq f(x) \leq h(x)) 在 (x \to a) 的过程中成立，并且 (\lim_{x \to a} g(x) = \lim_{x \to a} h(x) = A)，那么 (\lim_{x \to a} f(x) = A)。
  • 单调有界原理：单调有界数列必有极限。

五、两个重要极限

  • 重要极限Ⅰ：(\lim_{x \to 0} \frac{\sin(x)}{x} = 1)
  • 重要极限Ⅱ：(\lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^x = e)

第三节 函数的连续性

一、函数的连续性

  • 连续的定义：设函数 (y = f(x)) 在点 (x_0) 的某个邻域内有定义，如果 (\lim_{x \to x_0} f(x) = f(x_0))，则称函数 (f(x)) 在点 (x_0) 处连续。
  • 连续函数的性质：
    • 连续函数在图像上是一条连续无间断点的曲线。
    • 初等函数在其定义区间内都是连续的。

二、连续性的判定

  • 左连续：如果函数 ( f(x) ) 满足 (\lim_{x \to x_0^-} f(x) = f(x_0))，则称函数 ( f(x) ) 在点 ( x_0 ) 处左连续。
  • 右连续：如果函数 ( f(x) ) 满足 (\lim_{x \to x_0^+} f(x) = f(x_0))，则称函数 ( f(x) ) 在点 ( x_0 ) 处右连续。
  • 连续性判定定理：函数 ( f(x) ) 在点 ( x_0 ) 处连续的充分必要条件是 ( f(x) ) 在 ( x_0 ) 处既左连续又右连续。

三、闭区间上连续函数的性质

  • 最值定理：在闭区间 ([a, b]) 上连续的函数，一定能取得它的最大值和最小值。
  • 介值定理：设函数 ( f(x) ) 在闭区间 ([a, b]) 上连续，且最大值 ( M ) 不等于最小值 ( m )，那么对介于 ( m ) 与 ( M ) 之间的任意数 ( C )，在开区间 ((a, b)) 内至少存在一点 ( \xi )，使得 ( f(\xi) = C )。
  • 零点存在定理：设函数 ( f(x) ) 在闭区间 ([a, b]) 上连续，且 ( f(a) ) 与 ( f(b) ) 异号，那么在开区间 ((a, b)) 内至少存在一点 ( \xi )，使得 ( f(\xi) = 0 )。

第四节 导数

一、导数的概念

  • 导数的定义：设函数 ( y = f(x) ) 在点 ( x_0 ) 的某个邻域内有定义，当自变量 ( x ) 在 ( x_0 ) 处取得增量 ( \Delta x )（点 ( x_0 + \Delta x ) 仍在该邻域内）时，相应地函数 ( y ) 取得增量 ( \Delta y = f(x_0 + \Delta x) - f(x_0) )，若 (\frac{\Delta y}{\Delta x}) 当 ( \Delta x \to 0 ) 的极限存在，则称函数 ( y = f(x) ) 在点 ( x_0 ) 处可导，并称这个极限为函数 ( y = f(x) ) 在点 ( x_0 ) 处的导数，记为 ( f'(x_0) ) 或 (\frac{dy}{dx}\bigg|_{x=x_0})。
  • 导数的几何意义：函数 ( f(x) ) 在点 ( x_0 ) 处的导数 ( f'(x_0) ) 在几何上表示曲线 ( y = f(x) ) 在点 ( (x_0, f(x_0)) ) 处的切线的斜率。

二、导数的求法

  • 基本导数公式：
    • ((c)' = 0)，其中 ( c ) 为常数
    • ((x^\mu)' = \mu x^{\mu-1})
    • ((\sin(x))' = \cos(x))
    • ((\cos(x))' = -\sin(x))
    • ((\tan(x))' = \sec^2(x))
    • ((\cot(x))' = -\csc^2(x))
    • ((\sec(x))' = \sec(x)\tan(x))
    • ((\csc(x))' = -\csc(x)\cot(x))
    • ((a^x)' = a^x \ln(a))
    • ((e^x)' = e^x)
    • ((\log_a(x))' = \frac{1}{x \ln(a)})
    • ((\ln(x))' = \frac{1}{x})
    • ((\arcsin(x))' = \frac{1}{\sqrt{1-x^2}})
    • ((\arccos(x))' = -\frac{1}{\sqrt{1-x^2}})
    • ((\arctan(x))' = \frac{1}{1+x^2})
    • ((\text{arccot}(x))' = -\frac{1}{1+x^2})
  • 导数的四则运算法则：
    • ((u \pm v)' = u' \pm v')
    • ((uv)' = u'v + uv')
    • (\left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2})（( v \neq 0 )）
    • ((Cu)' = Cu')，其中 ( C ) 为常数
  • 复合函数的求导法则（链式法则）：如果函数 ( y = f(u) ) 在 ( u = g(x) ) 处可导，且 ( u = g(x) ) 在 ( x ) 处可导，则复合函数 ( y = f(g(x)) ) 在 ( x ) 处可导，且其导数为 ( \frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx} )。
  • 隐函数的求导：由方程 ( F(x, y) = 0 ) 确定的隐函数 ( y = f(x) ) 的导数，可以通过将方程两边同时对 ( x ) 求导得到。例如，对于方程 ( e^y + xy = e )，求导得 ( e^y \frac{dy}{dx} + y + x \frac{dy}{dx} = 0 )，解得 ( \frac{dy}{dx} = -\frac{y}{e^y + x} )。
  • 对数求导法：适用于幂指函数和以乘、除、乘方、开方运算为主的函数的求导。例如，对于函数 ( y = x^{x^2} )，取对数得 ( \ln(y) = x^2 \ln(x) )，对两边求导得 ( \frac{1}{y} \frac{dy}{dx} = 2x \ln(x) + x )，解得 ( \frac{dy}{dx} = y(2x \ln(x) + x) = x^{x^2}(2x \ln(x) + x) )。
  • 反函数的求导法则：如果函数 ( y = f(x) ) 在点 ( x_0 ) 处的导数存在，且 ( f'(x_0) \neq 0 )，在点 ( x_0 ) 的某一邻域内连续，且严格单调，则其反函数 ( x = \varphi(y) ) 在对应的点 ( y_0 ) 处可导，且 ( \varphi'(y_0) = \frac{1}{f'(x_0)} )。

三、高阶导数

  • 定义：函数 ( y = f(x) ) 的导数 ( y' = f'(x) ) 仍然是 ( x ) 的函数，我们把 ( f'(x) ) 的导数叫做 ( f(x) ) 的二阶导数，记作 ( f''(x) ) 或 ( \frac{d^2y}{dx^2} )。类似地，二阶导数的导数称为函数的三阶导数，记作 ( f'''(x) ) 或 ( \frac{d^3y}{dx^3} )，以此类推。
  • 例题：对于函数 ( y = \sin(x) )，求其 ( n ) 阶导数。解：( y' = \cos(x) )，( y'' = -\sin(x) )，( y''' = -\cos(x) )，( y^{(4)} = \sin(x) )，以此类推，每四阶导数重复一次，正弦、余弦交替出现。因此，( y^{(n)} = \sin\left(x + \frac{n\pi}{2}\right) )。

第五节 微分

一、微分的概念

  • 定义：设函数 ( y = f(x) ) 在点 ( x_0 ) 的邻域内有定义，给 ( x ) 以增量 ( \Delta x )（( x_0 + \Delta x )仍在该邻域内），若函数增量 ( \Delta y = f(x_0 + \Delta x) - f(x_0) ) 可表示为 ( \Delta y = A \Delta x + o(\Delta x) )，其中 ( A ) 是不依赖于 ( \Delta x ) 的常数，( o(\Delta x) ) 是比 ( \Delta x ) 高阶的无穷小，则称 ( A \Delta x ) 为函数 ( y = f(x) ) 在点 ( x_0 ) 处的微分，记作 ( dy ) 或 ( df(x_0) )，即 ( dy = A \Delta x )。此时，也称函数 ( y = f(x) ) 在点 ( x_0 ) 处可微。

二、可微与可导的关系

  • 定理：函数 ( f(x) ) 在点 ( x_0 ) 处可微的充分必要条件是 ( f(x) ) 在点 ( x_0 ) 处可导，且 ( dy = f'(x_0) \Delta x )。
  • 证明：
    • 可微⇒可导：如果函数 ( f(x) ) 在点 ( x_0 ) 处可微，则函数增量 ( \Delta y = A \Delta x + o(\Delta x) )，所以 ( \frac{\Delta y}{\Delta x} = A + \frac{o(\Delta x)}{\Delta x} )。当 ( \Delta x \to 0 ) 时，( \frac{o(\Delta x)}{\Delta x} \to 0 )，因此 ( \lim_{\Delta x \to 0} \frac{\Delta y}{\Delta x} = A )，即 ( f'(x_0) = A )。
    • 可导⇒可微：如果函数 ( f(x) ) 在点 ( x_0 ) 处可导，则 ( \lim_{\Delta x \to 0} \frac{\Delta y}{\Delta x} = f'(x_0) )，所以 ( \Delta y = f'(x_0) \Delta x + o(\Delta x) )，即 ( dy = f'(x_0) \Delta x )。

三、基本微分公式

  • 常数的微分：( dc = 0 )
  • 幂函数的微分：( d(x^\mu) = \mu x^{\mu-1} dx )
  • 指数函数的微分：
    • ( d(a^x) = a^x \ln(a) dx )
    • ( d(e^x) = e^x dx )
  • 对数函数的微分：
    • ( d(\log_a(x)) = \frac{1}{x \ln(a)} dx )
    • ( d(\ln(x)) = \frac{1}{x} dx )
  • 三角函数的微分：
    • ( d(\sin(x)) = \cos(x) dx )
    • ( d(\cos(x)) = -\sin(x) dx )
    • ( d(\tan(x)) = \sec^2(x) dx )
    • ( d(\cot(x)) = -\csc^2(x) dx )
    • ( d(\sec(x)) = \sec(x)\tan(x) dx )
    • ( d(\csc(x)) = -\csc(x)\cot(x) dx )
  • 反三角函数的微分：
    • ( d(\arcsin(x)) = \frac{1}{\sqrt{1-x^2}} dx )
    • ( d(\arccos(x)) = -\frac{1}{\sqrt{1-x^2}} dx )
    • ( d(\arctan(x)) = \frac{1}{1+x^2} dx )
    • ( d(\text{arccot}(x)) = -\frac{1}{1+x^2} dx )

四、微分的运算法则

  • 四则运算法则：
    • ( d(u \pm v) = du \pm dv )
    • ( d(uv) = vdu + udv )
    • ( d\left(\frac{u}{v}\right) = \frac{vdu - udv}{v^2} )（( v \neq 0 )）
    • ( d(cu) = cdu )，其中 ( c ) 为常数
  • 复合函数的微分法则：如果函数 ( y = f(u) ) 和 ( u = g(x) ) 都可导，则复合函数 ( y = f(g(x)) ) 的微分为 ( dy = f'(u) g'(x) dx )。由于 ( du = g'(x) dx )，所以 ( dy = f'(u) du )，这表明微分形式具有不变性，即无论 ( u ) 是自变量还是中间变量，微分公式都成立。

五、微分的几何意义

  • 切线近似：在点 ( M(x_0, f(x_0)) ) 的附近，可以用切线段近似代替曲线段。切线的斜率为 ( f'(x_0) )，切线方程为 ( y - f(x_0) = f'(x_0)(x - x_0) )。微分 ( dy ) 就是曲线的切线上点的纵坐标的相应增量，即 ( dy = f'(x_0) \Delta x )。当 ( \Delta x ) 很小时，( \Delta y \approx dy )，可以用切线段近似代替曲线段，以直代曲。

六、例题

  1. 求函数的微分：求函数 ( y = x^2 ) 在点 ( x = 2 ) 处的微分。
    • 解：因为 ( dy = 2x dx )，所以当 ( x = 2 ) 时，( dy = 2 \cdot 2 dx = 4 dx )。
  2. 利用微分求近似值：求 ( \sqrt{4.02} ) 的近似值。
    • 解：设 ( y = \sqrt{x} )，则 ( dy = \frac{1}{2\sqrt{x}} dx )。取 ( x = 4 )，( dx = 0.02 )，则 ( dy = \frac{1}{2\sqrt{4}} \cdot 0.02 = 0.01 )。因此，( \sqrt{4.02} \approx \sqrt{4} + dy = 2 + 0.01 = 2.01 )。
  3. 求隐函数的导数：求由方程 ( x^2 + y^2 = 1 ) 确定的隐函数 ( y = f(x) ) 的导数。
    • 解：将方程两边同时微分，得 ( 2x dx + 2y dy = 0 )，解得 ( dy = -\frac{x}{y} dx )，所以 ( \frac{dy}{dx} = -\frac{x}{y} )。

第二章 微分中值定理与导数的应用

1. 中值定理

  • 罗尔定理
    • 条件：函数 ( f(x) ) 在闭区间 ([a,b]) 上连续，在开区间 ((a,b)) 内可导，且 ( f(a) = f(b) ) .
    • 结论：在 ((a,b)) 内至少存在一点 ( \xi ) ，使得 ( f'(\xi) = 0 ) .
    • 几何意义：连续曲线 ( y = f(x) ) 的弧 ( AB ) 除端点外处处具有不垂直 ( x ) 轴的切线，且两个端点的纵坐标相等，则曲线弧上至少存在一点 ( C ) ，使得曲线在该点处的切线是水平的.
  • 拉格朗日中值定理
    • 条件：函数 ( f(x) ) 在闭区间 ([a,b]) 上连续，在开区间 ((a,b)) 内可导.
    • 结论：在 ((a,b)) 内至少存在一点 ( \xi ) ，使得 ( f(b) - f(a) = f'(\xi)(b - a) ) .
    • 几何意义：连续曲线 ( y = f(x) ) 的弧 ( AB ) 除端点外处处有不垂直 ( x ) 轴的切线，则弧上至少存在一点 ( \xi ) ，使得曲线在点 ( \xi ) 处的切线平行弦 ( AB ) .
  • 推论：如果函数 ( f(x) ) 在区间 ( I ) 上的导数恒为零，那么 ( f(x) ) 在区间 ( I ) 上是一个常数.
  • 重要性：拉格朗日中值定理公式 ( f(x+\Delta x) - f(x) = f'(\xi)\Delta x ) 给出了当自变量取得有限增量 ( \Delta x ) 时，函数增量 ( \Delta y ) 的准确表达式，也称为微分中值定理.

2. 洛必达法则

  • 基本形式
    • 条件：当 ( x \to a ) 时，( f(x) ) 和 ( g(x) ) 都趋于零；在点 ( a ) 的某去心邻域内 ( f'(x) ) 和 ( g'(x) ) 都存在且 ( g'(x) \neq 0 ) ；( \lim_{x \to a} \frac{f'(x)}{g'(x)} ) 存在或为 ( \infty ) .
    • 结论：( \lim_{x \to a} \frac{f(x)}{g(x)} = \lim_{x \to a} \frac{f'(x)}{g'(x)} ) .
  • 推广形式
    • 当 ( x \to \infty ) 时，洛必达法则也成立.
    • 对于 ( \frac{\infty}{\infty} ) 型的极限定值，也有洛必达法则.
  • 失效情况：如果 ( \lim_{x \to a} \frac{f'(x)}{g'(x)} ) 不存在，也不是 ( \infty ) ，则法则失效.
  • 其它形式的未定式
    • ( 0 \cdot \infty ) 型：将未定式变形为 ( \frac{0}{0} ) 或 ( \frac{\infty}{\infty} ) 型.
    • ( \infty - \infty ) 型：将未定式变形为 ( \frac{1}{0} - \frac{1}{0} ) 型，通分合并后使用洛必达法则.
    • ( 0^0, 1^\infty, \infty^0 ) 型：将未定式先取自然对数、变形，再按情形 ( \frac{0}{0} ) 或 ( \frac{\infty}{\infty} ) 处理.

3. 导数的应用

  • 函数的单调性
    • 判别定理
      • 如果函数 ( f(x) ) 在区间 ((a,b)) 内有 ( f'(x) > 0 ) ，则函数在 ([a,b]) 上是单调递增的.
      • 如果函数 ( f(x) ) 在区间 ((a,b)) 内有 ( f'(x) < 0 ) ，则函数在 ([a,b]) 上是单调递减的.
    • 求单调区间的方法
      • 求函数的一阶导数.
      • 找出所有的驻点及一阶导数不存在的点.
      • 将上述点插入到定义域，分区间确定一阶导数的符号.
      • 根据单调性的判别定理，确定单调区间.
  • 函数的极值
    • 极值的概念：如果函数 ( f(x) ) 在点 ( x_0 ) 的某邻域内有定义，对于该邻域内任意异于 ( x_0 ) 点的 ( x ) ，都有 ( f(x) \leq f(x_0) ) ，则称 ( f(x_0) ) 为函数的一个极小值；如果有 ( f(x) \geq f(x_0) ) ，则称 ( f(x_0) ) 为函数的一个极大值.极大值和极小值统称为函数的极值.使函数取得极值的点称为函数的极值点.
    • 极值存在的必要条件（费马定理）：如果函数 ( f(x) ) 在点 ( x_0 ) 处可导，且在点 ( x_0 ) 处有极值，则 ( f'(x_0) = 0 ) .导数为零的点称为函数的驻点.
    • 极值存在的第一充分条件
      • 如果在 ( x_0 ) 的某个邻域内，当 ( x < x_0 ) 时 ( f'(x) > 0 ) ，当 ( x > x_0 ) 时 ( f'(x) < 0 ) ，则 ( f(x) ) 在 ( x_0 ) 处取得极大值.
      • 如果在 ( x_0 ) 的某个邻域内，当 ( x < x_0 ) 时 ( f'(x) < 0 ) ，当 ( x > x_0 ) 时 ( f'(x) > 0 ) ，则 ( f(x) ) 在 ( x_0 ) 处取得极小值.
      • 如果在 ( x_0 ) 的某个邻域内，( f'(x) ) 同号，则 ( f(x) ) 在 ( x_0 ) 处无极值.
    • 极值存在的第二充分条件：设函数 ( f(x) ) 在点 ( x_0 ) 处具有二阶导数，且 ( f'(x_0) = 0 ) ，( f''(x_0) \neq 0 ) ，则当 ( f''(x_0) > 0 ) 时，( f(x_0) ) 为 ( f(x) ) 的极小值；当 ( f''(x_0) < 0 ) 时，( f(x_0) ) 为 ( f(x) ) 的极大值.
  • 函数的最大值与最小值
    • 求最值的步骤
      • 求函数的导数.
      • 在给定区间（或定义域）内找出所有的驻点及一阶导数不存在的点.
      • 计算函数在上述点处的函数值，以及在端点处的函数值，并比较其大小，其中最大者即为函数在区间上的最大值；最小者即为函数在区间上的最小值.
  • 曲线的凹凸向及拐点
    • 定义：如果曲线弧总位于它的每一点的切线的上方，则称该曲线弧是（向上）凹的；如果曲线弧总位于它的每一点的切线的下方，则称该曲线弧是（向上）凸的.凹、凸弧的分界点，称为曲线的拐点.
    • 凹凸弧的判别定理：设函数 ( f(x) ) 在区间 ((a,b)) 上具有二阶导数，则在该区间上：
      • 当 ( f''(x) > 0 ) 时，曲线弧是向上凹的.
      • 当 ( f''(x) < 0 )时，曲线弧是向上凸的.

4. 题目

  • 例 1：证明 ( \arctan x + arccot x = \frac{\pi}{2} ) ，( \forall x \in (-\infty, +\infty) ) .
    • 证明：令 ( f(x) = \arctan x + arccot x ) ，则 ( f(x) ) 在 ( (-\infty, +\infty) ) 内满足拉格朗日中值定理.而 ( f'(x) = \frac{1}{1+x^2} - \frac{1}{1+x^2} = 0 ) ，( \forall x \in (-\infty, +\infty) ) ，所以 ( f(x) \equiv C ) （常数）.而 ( f(1) = \arctan 1 + arccot 1 = \frac{\pi}{4} + \frac{\pi}{4} = \frac{\pi}{2} ) ，所以 ( f(x) = \arctan x + arccot x = \frac{\pi}{2} ) ，( \forall x \in (-\infty, +\infty) ) .
  • 例 2：证明当 ( x > 0 ) 时，( \ln(1+x) < x ) .
    • 证明：令 ( f(x) = \ln(1+x) ) ，则 ( f(x) ) 在区间 ([0, x]) 上满足拉格朗日中值定理条件.根据拉格朗日中值定理，有 ( \ln(1+x) - \ln(1+0) = f'(\xi)(x - 0) ) ，其中 ( \xi \in (0, x) ) ，即 ( \ln(1+x) = \frac{1}{1+\xi} \cdot x ) ，( \xi \in (0, x) ) .由于 ( 0 < \xi < x ) ，所以 ( 1 < 1+\xi < 1+x ) ，从而 ( \frac{1}{1+x} < \frac{1}{1+\xi} < 1 ) ，因此 ( \ln(1+x) = \frac{1}{1+\xi} \cdot x < x ) ，即当 ( x > 0 ) 时，( \ln(1+x) < x ) .
  • 例 3：求极限 ( \lim_{x \to 0} \frac{e^x - 1}{x} ) .
    • 解：这是 ( \frac{0}{0} ) 型的未定式，且当 ( x \to 0 ) 时，分子分母都趋于零，可以使用洛必达法则.对分子分母同时求导，得到 ( \lim_{x \to 0} \frac{e^x - 1}{x} = \lim_{x \to 0} \frac{e^x}{1} = e^0 = 1 ) .
  • 例 4：求函数 ( y = x^3 - 3x^2 - 9x + 2 ) 的单调区间和极值.
    • 解：首先求导数 ( y' = 3x^2 - 6x - 9 ) ，令 ( y' = 0 ) ，解得驻点 ( x = -1 ) 和 ( x = 3 ) .将驻点和一阶导数不存在的点（本题中不存在）插入到定义域，分区间确定一阶导数的符号，得到 ( y' > 0 ) 在区间 ( (-\infty, -1) ) 和 ( (3, +\infty) ) 上，( y' < 0 ) 在区间 ( (-1, 3) ) 上.根据单调性的判别定理，函数在 ( (-\infty, -1) ) 和 ( (3, +\infty) ) 上单调递增，在 ( (-1, 3) ) 上单调递减.因此，函数在 ( x = -1 ) 处取得极大值 ( y(-1) = 7 ) ，在 ( x = 3 ) 处取得极小值 ( y(3) = -25 ) .
  • 例 5：求曲线 ( y = \frac{1}{1+x^2} ) 的凹凸区间及拐点.
    • 解：首先求导数 ( y' = -\frac{2x}{(1+x^2)^2} ) ，再求二阶导数 ( y'' = \frac{2(3x^2 - 1)}{(1+x^2)^3} ) .令 ( y'' = 0 ) ，解得 ( x = \pm \frac{\sqrt{3}}{3} ) .将这些点插入到定义域，分区间确定二阶导数的符号，得到 ( y'' > 0 ) 在区间 ( (-\infty, -\frac{\sqrt{3}}{3}) ) 和 ( (\frac{\sqrt{3}}{3}, +\infty) ) 上，( y'' < 0 ) 在区间 ( (-\frac{\sqrt{3}}{3}, \frac{\sqrt{3}}{3}) ) 上.根据凹凸弧的判别定理，曲线在 ( (-\infty, -\frac{\sqrt{3}}{3}) ) 和 ( (\frac{\sqrt{3}}{3}, +\infty) ) 上是向上凹的，在 ( (-\frac{\sqrt{3}}{3}, \frac{\sqrt{3}}{3}) ) 上是向上凸的，有拐点 ( (-\frac{\sqrt{3}}{3}, \frac{3}{4}) ) 和 ( (\frac{\sqrt{3}}{3}, \frac{3}{4}) ) .

第四章 一元函数积分

第一节 不定积分

学习重点

  • 原函数的概念及不定积分的概念
  • 不定积分的计算

原函数的概念

  • 定义：如果在区间 ( I ) 内的每一点处，有 ( F'(x) = f(x) )，则称 ( F(x) ) 是 ( f(x) ) 在区间 ( I ) 内的一个原函数.
  • 性质：
    • 如果 ( F(x) ) 是 ( f(x) ) 的一个原函数，则 ( F(x) + C ) 也是 ( f(x) ) 的原函数，其中 ( C ) 是任意常数.
    • 如果函数 ( f(x) ) 在区间 ( I ) 内连续，则函数 ( f(x) ) 在该区间内一定有原函数.

不定积分的概念

  • 定义：函数 ( f(x) ) 在区间 ( I ) 内的所有原函数构成的集合，称为函数 ( f(x) ) 在区间 ( I ) 内的不定积分，记作 ( \int f(x) , dx ).
  • 基本性质：
    • ( \int f(x) , dx = F(x) + C )，其中 ( F(x) ) 是 ( f(x) ) 的一个原函数，( C ) 是积分常数.
    • ( \int [f(x) \pm g(x)] , dx = \int f(x) , dx \pm \int g(x) , dx ).
    • ( \int k f(x) , dx = k \int f(x) , dx )，其中 ( k ) 是常数.

基本积分表

  • ( \int x^\mu , dx = \frac{x^{\mu+1}}{\mu+1} + C ) （( \mu \neq -1 )）
  • ( \int \frac{1}{x} , dx = \ln |x| + C )
  • ( \int e^x , dx = e^x + C )
  • ( \int a^x , dx = \frac{a^x}{\ln a} + C )
  • ( \int \frac{1}{1+x^2} , dx = \arctan x + C )
  • ( \int \frac{1}{\sqrt{1-x^2}} , dx = \arcsin x + C )
  • ( \int \cos x , dx = \sin x + C )
  • ( \int \sin x , dx = -\cos x + C )
  • ( \int \sec^2 x , dx = \tan x + C )
  • ( \int \csc^2 x , dx = -\cot x + C )
  • ( \int \sec x \tan x , dx = \sec x + C )
  • ( \int \csc x \cot x , dx = -\csc x + C )

不定积分的计算方法

  • 直接积分法：利用基本积分公式和性质直接计算.
  • 换元积分法：
    • 第一类换元积分法：令 ( u = \varphi(x) )，则 ( \int f[\varphi(x)] \varphi'(x) , dx = \int f(u) , du ).
    • 第二类换元积分法：令 ( x = \varphi(u) )，则 ( \int f(x) , dx = \int f[\varphi(u)] \varphi'(u) , du ).
  • 分部积分法：( \int u , dv = uv - \int v , du ).

例题

  • 求 ( \int \frac{1}{1+x^2} , dx )：由基本积分公式，得 ( \int \frac{1}{1+x^2} , dx = \arctan x + C ).
  • 求 ( \int x^2 , dx )：由基本积分公式，得 ( \int x^2 , dx = \frac{x^3}{3} + C ).
  • 求 ( \int \sin 2x , dx )：令 ( u = 2x )，则 ( du = 2 , dx )，所以 ( \int \sin 2x , dx = \frac{1}{2} \int \sin u , du = -\frac{1}{2} \cos u + C = -\frac{1}{2} \cos 2x + C ).

第二节 定积分

定积分的概念

  • 定义：设函数 ( f(x) ) 在区间 ([a, b]) 上有定义，将区间 ([a, b]) 分成 ( n ) 个子区间 ([x_{i-1}, x_i])，在每个子区间上任取一点 ( \xi_i )，作和 ( \sum_{i=1}^{n} f(\xi_i) \Delta x_i )，当 ( \lambda \to 0 ) 时，如果这个和的极限存在，则称函数 ( f(x) ) 在区间 ([a, b]) 上可积，这个极限称为函数 ( f(x) ) 在区间 ([a, b]) 上的定积分，记作 ( \int_{a}^{b} f(x) , dx ).

定积分存在的充分条件

  • 函数 ( f(x) ) 在区间 ([a, b]) 上连续.
  • 函数 ( f(x) ) 在区间 ([a, b]) 上有界，且只有有限个间断点.

定积分的几何意义

  • 定积分 ( \int_{a}^{b} f(x) , dx ) 表示曲线 ( y = f(x) ) 与 ( x ) 轴在区间 ([a, b]) 上所围成的图形的面积的代数和.

定积分的基本性质

  • ( \int_{a}^{b} f(x) , dx = -\int_{b}^{a} f(x) , dx )
  • ( \int_{a}^{b} [f(x) \pm g(x)] , dx = \int_{a}^{b} f(x) , dx \pm \int_{a}^{b} g(x) , dx )
  • ( \int_{a}^{b} k f(x) , dx = k \int_{a}^{b} f(x) , dx )，其中 ( k ) 是常数.
  • ( \int_{a}^{b} f(x) , dx = \int_{a}^{c} f(x) , dx + \int_{c}^{b} f(x) , dx )，其中 ( a \leq c \leq b ).
  • ( \int_{a}^{b} dx = b - a )
  • 如果 ( f(x) \geq g(x) ) 在区间 ([a, b]) 上成立，则 ( \int_{a}^{b} f(x) , dx \geq \int_{a}^{b} g(x) , dx ).
  • ( m(b-a) \leq \int_{a}^{b} f(x) , dx \leq M(b-a) )，其中 ( m ) 和 ( M ) 分别是函数 ( f(x) ) 在区间 ([a, b]) 上的最小值和最大值.

微积分基本公式

  • 牛顿-莱布尼兹公式：如果函数 ( f(x) ) 在区间 ([a, b]) 上连续，( F(x) ) 是 ( f(x) ) 的一个原函数，则 ( \int_{a}^{b} f(x) , dx = F(b) - F(a) ).

例题

  • 求 ( \int_{0}^{1} x^2 , dx )：由牛顿-莱布尼兹公式，得 ( \int_{0}^{1} x^2 , dx = \left. \frac{x^3}{3} \right|_{0}^{1} = \frac{1}{3} ).
  • 求 ( \int_{0}^{\pi} \sin x , dx )：由牛顿-莱布尼兹公式，得 ( \int_{0}^{\pi} \sin x , dx = \left. -\cos x \right|_{0}^{\pi} = 2 ).

第三节 广义积分

无穷区间上的广义积分

  • 定义：
  • ( \int_{a}^{+\infty} f(x) , dx = \lim_{b \to +\infty} \int_{a}^{b} f(x) , dx )
  • ( \int_{-\infty}^{b} f(x) , dx = \lim_{a \to -\infty} \int_{a}^{b} f(x) , dx )
  • ( \int_{-\infty}^{+\infty} f(x) , dx = \int_{-\infty}^{c} f(x) , dx + \int_{c}^{+\infty} f(x) , dx )，其中 ( c ) 是任意实数.
  • 敛散性：如果上述极限存在，则称广义积分收敛，否则称广义积分发散.

例题

  • 求 ( \int_{1}^{+\infty} \frac{1}{x^2} , dx )：由定义，得 ( \int_{1}^{+\infty} \frac{1}{x^2} , dx = \lim_{b \to +\infty} \int_{1}^{b} \frac{1}{x^2} , dx = \lim_{b \to +\infty} \left. -\frac{1}{x} \right|_{1}^{b} = 1 )，所以该广义积分收敛.
  • 求 ( \int_{-\infty}^{+\infty} \frac{1}{1+x^2} , dx )：由定义，得 ( \int_{-\infty}^{+\infty} \frac{1}{1+x^2} , dx = \int_{-\infty}^{0} \frac{1}{1+x^2} , dx + \int_{0}^{+\infty} \frac{1}{1+x^2} , dx = \lim_{a \to -\infty} \int_{a}^{0} \frac{1}{1+x^2} , dx + \lim_{b \to +\infty} \int_{0}^{b} \frac{1}{1+x^2} , dx = \lim_{a \to -\infty} \left. \arctan x \right|{a}^{0} + \lim{b \to +\infty} \left. \arctan x \right|_{0}^{b} = \frac{\pi}{2} + \frac{\pi}{2} = \pi )，所以该广义积分收敛.

第四节 定积分的计算

定积分的换元法

  • 基本换元规律：与不定积分相同，但必须做到换元同时换限.
  • 例题：
    • 求 ( \int_{0}^{1} \frac{1}{\sqrt{1-x^2}} , dx )：令 ( x = \sin t )，则 ( dx = \cos t , dt )，当 ( x = 0 ) 时，( t = 0 )；当 ( x = 1 ) 时，( t = \frac{\pi}{2} )，所以 ( \int_{0}^{1} \frac{1}{\sqrt{1-x^2}} , dx = \int_{0}^{\frac{\pi}{2}} \frac{\cos t}{\cos t} , dt = \int_{0}^{\frac{\pi}{2}} dt = \frac{\pi}{2} ).

定积分的分部积分法

  • 公式：( \int_{a}^{b} u , dv = \left. uv \right|{a}^{b} - \int{a}^{b} v , du ).
  • 例题：
    • 求 ( \int_{0}^{\pi} x \sin x , dx )：令 ( u = x )，( dv = \sin x , dx )，则 ( du = dx )，( v = -\cos x )，所以 ( \int_{0}^{\pi} x \sin x , dx = \left. -x \cos x \right|{0}^{\pi} - \int{0}^{\pi} -\cos x , dx = \pi + \left. \sin x \right|_{0}^{\pi} = \pi ).

第五节 定积分的应用

定积分在几何中的应用

  • 平面图形的面积：设平面图形由曲线 ( y = f(x) ) 和 ( y = g(x) ) 以及直线 ( x = a ) 和 ( x = b ) 所围成，其中 ( f(x) \geq g(x) ) 在区间 ([a, b]) 上成立，则该平面图形的面积 ( A ) 为 ( A = \int_{a}^{b} [f(x) - g(x)] , dx ).
  • 旋转体的体积：
    • 绕 ( x ) 轴旋转：设平面图形由曲线 ( y = f(x) ) 和 ( x ) 轴以及直线 ( x = a ) 和 ( x = b ) 所围成，则该平面图形绕 ( x ) 轴旋转一周所形成的旋转体的体积 ( V ) 为 ( V = \pi \int_{a}^{b} [f(x)]^2 , dx ).
    • 绕 ( y ) 轴旋转：设平面图形由曲线 ( x = f(y) ) 和 ( y ) 轴以及直线 ( y = c ) 和 ( y = d ) 所围成，则该平面图形绕 ( y ) 轴旋转一周所形成的旋转体的体积 ( V ) 为 ( V = \pi \int_{c}^{d} [f(y)]^2 , dy ).

定积分在物理中的应用

  • 变速直线运动的路程：设物体的运动速度为 ( v(t) )，则物体在时间区间 ([t_1, t_2]) 内所经过的路程 ( S ) 为 ( S = \int_{t_1}^{t_2} v(t) , dt ).
  • 变力做功：设物体在变力 ( F(x) ) 的作用下沿 ( x ) 轴从 ( x = a ) 移动到 ( x = b )，则该变力所做的功 ( W ) 为 ( W = \int_{a}^{b} F(x) , dx ).

第六节 定积分的性质与定理

定积分的估值定理

  • 定理：设函数 ( f(x) ) 在区间 ([a, b]) 上连续，( m ) 和 ( M ) 分别是函数 ( f(x) ) 在区间 ([a, b]) 上的最小值和最大值，则有 ( m(b-a) \leq \int_{a}^{b} f(x) , dx \leq M(b-a) ).

定积分的积分中值定理

  • 定理：设函数 ( f(x) ) 在区间 ([a, b]) 上连续，则存在 ( \xi \in [a, b] )，使得 ( \int_{a}^{b} f(x) , dx = f(\xi)(b-a) ).

第七节 定积分的计算技巧与方法

换元积分法

  • 第一类换元积分法：适用于被积函数中含有复合函数的情况，通过令复合函数的内函数为新变量，简化积分过程.
    • 例题：求 ( \int \frac{1}{\sqrt{1-x^2}} , dx ).
      • 令 ( x = \sin t )，则 ( dx = \cos t , dt )，积分变为 ( \int \frac{\cos t}{\cos t} , dt = \int dt = t + C = \arcsin x + C ).
  • 第二类换元积分法：适用于被积函数中含有根式的情况，通过令根式为新变量，将根式转化为有理式，便于积分.
    • 例题：求 ( \int \frac{1}{\sqrt{a^2-x^2}} , dx ).
      • 令 ( x = a \sin t )，则 ( dx = a \cos t , dt )，积分变为 ( \int \frac{a \cos t}{a \cos t} , dt = \int dt = t + C = \arcsin \frac{x}{a} + C ).

分部积分法

  • 公式：( \int u , dv = uv - \int v , du )，适用于被积函数为两个函数乘积的情况.
    • 例题：求 ( \int x e^x , dx ).
      • 令 ( u = x )，( dv = e^x , dx )，则 ( du = dx )，( v = e^x )，积分变为 ( x e^x - \int e^x , dx = x e^x - e^x + C ).

对称区间上的积分性质

  • 奇函数的积分性质：若 ( f(x) ) 是奇函数，则 ( \int_{-a}^{a} f(x) , dx = 0 ).
  • 偶函数的积分性质：若 ( f(x) ) 是偶函数，则 ( \int_{-a}^{a} f(x) , dx = 2 \int_{0}^{a} f(x) , dx ).
    • 例题：求 ( \int_{-\pi}^{\pi} \cos^2 x , dx ).
      • 因为 ( \cos^2 x ) 是偶函数，所以 ( \int_{-\pi}^{\pi} \cos^2 x , dx = 2 \int_{0}^{\pi} \cos^2 x , dx = 2 \int_{0}^{\pi} \frac{1+\cos 2x}{2} , dx = \pi ).

第五章 积分的应用及微分方程

第一节 不定积分的应用

  • 学习重点：微分方程的概念、一阶微分方程的求解
  • 微分方程的概念
    • 定义：含有未知函数的导数或微分的方程，如 (y''=y+\sin x)、(\frac{dy}{dx}=3) 等.
    • 特点：(x) 和 (y) 可以不出现，但 (y) 的导数一定要出现.
    • 微分方程的阶：微分方程中出现的未知函数的导数的最高阶数.上面三个微分方程的阶数分别是二阶、一阶、三阶.
    • 微分方程的解：满足微分方程的函数.
      • 特解：满足微分方程且不含任意常数的函数.
      • 通解：满足 (n) 阶微分方程且含 (n) 个独立任意常数的函数.

第三节 微分方程

  • 一阶微分方程的求解
    • 可分离变量的微分方程
      • 形式：(\frac{dy}{dx}=f(x)g(y)) 或 (f(x)dx=g(y)dy)
      • 求解方法：两边同时积分 (\int f(x)dx=\int g(y)dy)
        • 理由：设 (y=y(x)) 是该微分方程的解，则 (\int g(y)dy=\int g(y(x))dy(x))，(\int f(x)dx=\int g(y(x))y'(x)dx)，(\int f(x)dx=\int g(y)dy).
      • 特例：(\frac{dy}{dx}=f(x)) 情形，即 (dy=f(x)dx)，两边积分，得 (y=\int f(x)dx).
  • 例题 - 求下列微分方程的通解或特解： - (y'+\frac{1}{10x}y=\frac{1}{10}) - 解：原方程可变形为 (\frac{dy}{dx}+\frac{1}{10x}y=\frac{1}{10})，分离变量得 (\frac{dy}{dx}=\frac{1}{10}-\frac{1}{10x}y)，即 (\frac{dy}{dx}=\frac{1}{10}(1-\frac{y}{x}))，两边积分得 (\int\frac{dy}{1-\frac{y}{x}}=\int\frac{1}{10}dx)，令 (u=1-\frac{y}{x})，则 (du=-\frac{1}{x}dy)，(\int\frac{-x}{u}du=\int\frac{1}{10}dx)，(-x\ln|u|=\frac{x}{10}+C)，(\ln|1-\frac{y}{x}|=-\frac{1}{10}+\frac{C}{x})，(1-\frac{y}{x}=e^{-\frac{1}{10}+\frac{C}{x}})，(y=x(1-e^{-\frac{1}{10}+\frac{C}{x}})). - (y'=\frac{xy^2}{x^2+y^2}) - 解：原方程可变形为 (\frac{dy}{dx}=\frac{xy^2}{x^2+y^2})，分离变量得 (\frac{dy}{y^2}=\frac{x}{x^2+y^2}dx)，两边积分得 (\int\frac{dy}{y^2}=\int\frac{x}{x^2+y^2}dx)，(-\frac{1}{y}=\frac{1}{2}\ln(x^2+y^2)+C)，(y=-\frac{2}{\ln(x^2+y^2)+2C}). - ((x^2+y^2)dx+2xydy=0) - 解：原方程可变形为 (\frac{dy}{dx}=-\frac{x^2+y^2}{2xy})，分离变量得 (\frac{dy}{dx}=-\frac{x}{2y}-\frac{y}{2x})，两边积分得 (\int\frac{dy}{y}=-\int\frac{x}{2}dx-\int\frac{y}{2x}dx)，(\ln|y|=-\frac{x^2}{4}-\frac{y^2}{4x^2}+C)，(y=e^{-\frac{x^2}{4}-\frac{y^2}{4x^2}+C}).
    • 一阶线性微分方程
      • 形式：(y'+p(x)y=q(x))
      • 求解方法
        • 齐次方程：当 (q(x)=0) 时，方程为 (y'+p(x)y=0)，可分离变量得 (\frac{dy}{y}=-p(x)dx)，两边积分得 (\ln|y|=-\int p(x)dx+C)，(y=Ce^{-\int p(x)dx}).
        • 非齐次方程：当 (q(x)\neq 0) 时，可采用常数变易法求解.设 (y=C(x)e^{-\int p(x)dx})，代入原方程得 (C'(x)e^{-\int p(x)dx}+C(x)(-p(x))e^{-\int p(x)dx}+p(x)C(x)e^{-\int p(x)dx}=q(x))，化简得 (C'(x)=q(x)e^{\int p(x)dx})，两边积分得 (C(x)=\int q(x)e^{\int p(x)dx}dx+C_1)，代回得 (y=e^{-\int p(x)dx}\left(\int q(x)e^{\int p(x)dx}dx+C_1\right)).
  • 例题 - (y'+\tan x\cdot y=\sin x) - 解：这是一个一阶线性微分方程，方程的通解为 (y=e^{-\int\tan xdx}\left(\int\sin xe^{\int\tan xdx}dx+C\right))，(\int\tan xdx=\ln|\sec x|)，(y=\cos x\left(\int\sin x\sec xdx+C\right))，(\int\sin x\sec xdx=\int\tan xdx=\ln|\sec x|)，(y=\cos x(\ln|\sec x|+C)). - (y'-\frac{2}{x}y=\frac{2}{x^2}) - 解：将原方程化为 (y'-\frac{2}{x}y=\frac{2}{x^2})，这是一个一阶线性微分方程，方程的通解为 (y=e^{\int\frac{2}{x}dx}\left(\int\frac{2}{x^2}e^{-\int\frac{2}{x}dx}dx+C\right))，(\int\frac{2}{x}dx=2\ln|x|)，(y=x^2\left(\int\frac{2}{x^2}\cdot\frac{1}{x^2}dx+C\right))，(\int\frac{2}{x^4}dx=-\frac{2}{3x^3})，(y=x^2\left(-\frac{2}{3x^3}+C\right))，(y=-\frac{2}{3x}+Cx^2). - (y'+\frac{1}{x}y=\frac{e^x}{x}) - 解：将原方程化为 (y'+\frac{1}{x}y=\frac{e^x}{x})，这是一个一阶线性微分方程，方程的通解为 (y=e^{-\int\frac{1}{x}dx}\left(\int\frac{e^x}{x}e^{\int\frac{1}{x}dx}dx+C\right))，(\int\frac{1}{x}dx=\ln|x|)，(y=\frac{1}{x}\left(\int\frac{e^x}{x}\cdot xdx+C\right))，(\int e^xdx=e^x)，(y=\frac{1}{x}(e^x+C)).

多元函数微积分

空间解析几何简介

  • 邻域：平面点集 ( U(P_0, \delta) ) 称为点 ( P_0(x_0, y_0) ) 的 ( \delta ) 邻域，满足 ( (x - x_0)^2 + (y - y_0)^2 \leq \delta^2 ) 。
  • 内点：若存在点 ( P ) 的某一邻域 ( U(P) ) 使得 ( U(P) \subseteq E ) ，则称 ( P ) 为 ( E ) 的内点。
  • 边界点：若点 ( P ) 的任意邻域 ( U(P) ) 都有属于 ( E ) 中的点，也有不属于 ( E ) 的点，则称 ( P ) 为 ( E ) 的边界点。
  • 开集：点集 ( E ) 中的点都是内点，则称 ( E ) 为开集。
  • 连通集：点集 ( E ) 中的任意两点，都可以用完全属于 ( E ) 中的折线段将它们连接起来，则称 ( E ) 为连通集。
  • 区域：连通的开集称为开区域，简称区域。
  • 闭区域：区域连同它的边界，称为闭区域。

二元函数的概念

  • 定义：设 ( D ) 是平面上的非空点集，如果存在一个对应法则 ( f ) ，使得对集合 ( D ) 中的每一个点 ( (x, y) ) ，按法则 ( f ) ，都有唯一确定的实数值 ( z ) 与之对应，则称此对应法则 ( f ) 为集合 ( D ) 上的二元函数，记为 ( f: (x, y) \rightarrow z ) 或 ( z = f(x, y) ) ，( (x, y) \in D ) 。
  • 图像：空间点集 ( {(x, y, z) | z = f(x, y), (x, y) \in D} ) 称为函数 ( z = f(x, y) ) 的图像，表示空间曲面。

二元函数的极限

  • 定义：设二元函数 ( z = f(x, y) ) 在点 ( P_0(x_0, y_0) ) 的邻域内有定义（点 ( P_0 ) 可以除外），如果当点 ( P(x, y) ) 无论以何种方式趋向于点 ( P_0(x_0, y_0) ) 时，函数值 ( f(x, y) ) 可以无限逼近常数 ( A ) ，则称 ( A ) 为函数 ( f(x, y) ) 在 ( P \rightarrow P_0 ) 时的极限，记作 ( \lim_{(x, y) \rightarrow (x_0, y_0)} f(x, y) = A ) 。

二元函数的连续性

  • 定义：若 ( \lim_{(x, y) \rightarrow (x_0, y_0)} f(x, y) = f(x_0, y_0) ) ，则称函数 ( f(x, y) ) 在点 ( (x_0, y_0) ) 处连续。若函数在某区域上点点连续，则称函数在该区域上连续。

偏导数和全微分

  • 偏导数的定义：设函数 ( z = f(x, y) ) 在点 ( (x_0, y_0) ) 的某一邻域内有定义，若极限 ( \lim_{\Delta x \rightarrow 0} \frac{f(x_0 + \Delta x, y_0) - f(x_0, y_0)}{\Delta x} ) 存在，则称此极限为函数 ( z = f(x, y) ) 在点 ( (x_0, y_0) ) 处对 ( x ) 的偏导数，记作 ( \frac{\partial z}{\partial x} ) 或 ( f_x(x_0, y_0) ) ；同理，对 ( y ) 的偏导数记作 ( \frac{\partial z}{\partial y} ) 或 ( f_y(x_0, y_0) ) 。
  • 全微分：若二元函数 ( z = f(x, y) ) 在点 ( (x, y) ) 处的全增量 ( \Delta z = f(x + \Delta x, y + \Delta y) - f(x, y) ) 可以表示为 ( \Delta z = A \Delta x + B \Delta y + o(\sqrt{\Delta x^2 + \Delta y^2}) ) ，则称函数在该点处可微，并称 ( dz = A \Delta x + B \Delta y ) 为函数在该点处的全微分，其中 ( A = \frac{\partial z}{\partial x} ) ，( B = \frac{\partial z}{\partial y} ) 。

多元复合函数与隐函数的微分法

  • 复合函数的求导法则（链式法则）：
    • 对于复合函数 ( z = f(u, v) ) ，( u = u(x, y) ) ，( v = v(x, y) ) ，有 ( \frac{\partial z}{\partial x} = \frac{\partial z}{\partial u} \frac{\partial u}{\partial x} + \frac{\partial z}{\partial v} \frac{\partial v}{\partial x} ) ，( \frac{\partial z}{\partial y} = \frac{\partial z}{\partial u} \frac{\partial u}{\partial y} + \frac{\partial z}{\partial v} \frac{\partial v}{\partial y} ) 。
  • 隐函数的求导问题：
    • 若方程 ( F(x, y) = 0 ) 确定了函数 ( y = f(x) ) ，则 ( \frac{dy}{dx} = -\frac{F_x}{F_y} ) 。

多元函数的极值

  • 极值存在的必要条件：各偏导存在的极值点一定是驻点（使各偏导数均为零的点）。
  • 极值存在的充分条件（以二元函数为例）：
    • 设函数 ( z = f(x, y) ) 在点 ( (x_0, y_0) ) 的某个邻域内连续且有直到二阶的连续偏导数，又 ( (x_0, y_0) ) 是驻点，令 ( A = f_{xx}(x_0, y_0) ) ，( B = f_{xy}(x_0, y_0) ) ，( C = f_{yy}(x_0, y_0) ) ，则：
      • 当 ( AC - B^2 > 0 ) 时，函数取到极值，且当 ( A > 0 ) 时取极小值，当 ( A < 0 ) 时取极大值。
      • 当 ( AC - B^2 < 0 ) 时，函数取不到极值。
      • 当 ( AC - B^2 = 0 ) 时，函数可能取到也可能取不到极值。

二重积分

  • 概念：设函数 ( z = f(x, y) ) 是有界闭区域 ( D ) 上的有界函数，将闭区域 ( D ) 任意分成 ( n ) 个小闭区域 ( \Delta \sigma_1, \Delta \sigma_2, \ldots, \Delta \sigma_n ) ，在每个小闭区域 ( \Delta \sigma_i ) 上任取一点 ( (\xi_i, \eta_i) ) ，令 ( \lambda = \max {\Delta \sigma_i} ) ，若无论 ( D ) 如何划分和 ( (\xi_i, \eta_i) ) 如何选取，极限 ( \lim_{\lambda \rightarrow 0} \sum_{i=1}^{n} f(\xi_i, \eta_i) \Delta \sigma_i ) 都存在，则称此极限为函数 ( z = f(x, y) ) 在 ( D ) 上的二重积分，记作 ( \iint_D f(x, y) , d\sigma ) 。
  • 性质：
  • ( \iint_D [f(x, y) \pm g(x, y)] , d\sigma = \iint_D f(x, y) , d\sigma \pm \iint_D g(x, y) , d\sigma )
  • 若 ( D ) 可分割为 ( D_1 ) 和 ( D_2 ) ，则 ( \iint_D f(x, y) , d\sigma = \iint_{D_1} f(x, y) , d\sigma + \iint_{D_2} f(x, y) , d\sigma )
  • ( \iint_D d\sigma = S ) ，其中 ( S ) 为区域 ( D ) 的面积
  • 若 ( f(x, y) \leq g(x, y) ) ，则 ( \iint_D f(x, y) , d\sigma \leq \iint_D g(x, y) , d\sigma )
  • 若 ( m \leq f(x, y) \leq M ) ，则 ( mS \leq \iint_D f(x, y) , d\sigma \leq MS )
  • 积分中值定理：若 ( f(x, y) ) 在 ( D ) 上连续，则存在 ( (\xi, \eta) \in D ) 使得 ( \iint_D f(x, y) , d\sigma = f(\xi, \eta)S )

二重积分的计算

  • 直角坐标系下的计算：
    • X型区域：若积分区域 ( D ) 可表示为 ( a \leq x \leq b ) ，( y_1(x) \leq y \leq y_2(x) ) ，则 ( \iint_D f(x, y) , d\sigma = \int_a^b \int_{y_1(x)}^{y_2(x)} f(x, y) , dy , dx ) 。
    • Y型区域：若积分区域 ( D ) 可表示为 ( c \leq y \leq d ) ，( x_1(y) \leq x \leq x_2(y) ) ，则 ( \iint_D f(x, y) , d\sigma = \int_c^d \int_{x_1(y)}^{x_2(y)} f(x, y) , dx , dy ) 。
  • 交换积分次序：若积分区域 ( D ) 既可表示为X型区域又可表示为Y型区域，则 ( \iint_D f(x, y) , d\sigma ) 可以交换积分次序计算。

题目练习

  • 例1：计算二重积分 ( \iint_D (x^2 + y^2) , d\sigma ) ，其中 ( D ) 为 ( 1 \leq x \leq 2 ) ，( 0 \leq y \leq 1 ) 。
    • 解：( \iint_D (x^2 + y^2) , d\sigma = \int_1^2 \int_0^1 (x^2 + y^2) , dy , dx = \int_1^2 \left[ x^2y + \frac{y^3}{3} \right]_0^1 , dx = \int_1^2 \left( x^2 + \frac{1}{3} \right) , dx = \left[ \frac{x^3}{3} + \frac{x}{3} \right]_1^2 = \frac{8}{3} + \frac{2}{3} - \frac{1}{3} - \frac{1}{3} = \frac{8}{3} ) 。

二重积分的计算（续）

  • 极坐标下的计算：
    • 对于区域 ( D ) 为圆或圆的一部分，使用极坐标变换 ( x = r\cos\theta ) ，( y = r\sin\theta ) ，( d\sigma = r , dr , d\theta ) 。
    • 例2续：计算二重积分 ( \iint_D xy , d\sigma ) ，其中 ( D ) 为 ( x^2 + y^2 \leq 1 ) 。
      • 解：在极坐标下，( xy = r^2\cos\theta\sin\theta ) ，积分区域 ( D ) 变为 ( 0 \leq r \leq 1 ) ，( 0 \leq \theta \leq 2\pi ) 。
      • 因此，( \iint_D xy , d\sigma = \int_0^{2\pi} \int_0^1 r^2\cos\theta\sin\theta \cdot r , dr , d\theta = \int_0^{2\pi} \cos\theta\sin\theta , d\theta \int_0^1 r^3 , dr ) 。
      • 计算得 ( \int_0^{2\pi} \cos\theta\sin\theta , d\theta = 0 ) （因为 ( \cos\theta\sin\theta ) 在一个周期内的积分为零），所以 ( \iint_D xy , d\sigma = 0 ) 。

高阶偏导数

  • 定义：由于二元函数的偏导数仍是二元函数，故可据实际需要再求偏导数，称之为二阶偏导数，同理有三阶、四阶等高阶偏导数。
  • 混合偏导数相等：若二元函数 ( z = f(x, y) ) 的两个混合偏导数 ( \frac{\partial^2 z}{\partial x \partial y} ) 和 ( \frac{\partial^2 z}{\partial y \partial x} ) 在区域 ( D ) 上连续，则它们必相等。

全微分、偏导数、连续性之间的关系

  • 连续性：若函数在某点连续，则在该点的偏导数存在。
  • 可微性：若函数在某点可微，则在该点的偏导数存在，且全微分存在。
  • 偏导存在：偏导数存在不一定意味着函数在该点连续或可微。

    • 解：首先计算一阶偏导数 ( \frac{\partial z}{\partial x} = 2xe^{x^2 + y^2} ) ，( \frac{\partial z}{\partial y} = 2ye^{x^2 + y^2} ) 。

第七章：行列式与矩阵基础

一、行列式的概念与性质

  • n阶行列式的定义：由n行n列元素构成，其值可通过展开式计算，展开式为各元素与其代数余子式的乘积之和。
  • 行列式的性质：
    • 转置行列式的值不变，即行列式与其转置行列式相等。$\text{det}(A^T) = \text{det}(A)$
    • 互换行列式的两行（列），行列式变号。$\text{det}(A') = -\text{det}(A)$
    • 某一行（列）的所有元素都乘以同一数k，等于用数k乘此行列式。$\text{det}(kA) = k^n \text{det}(A)$
    • 如果某一行（列）的元素都是两项的和，则可以把该行列式拆成两个行列式之和。$\text{det}(A+B) = \text{det}(A) + \text{det}(B)$
    • 把某一行（列）的元素都乘以同一个数k后，加到另一行（列）的对应元素上去，则行列式的值不变。$\text{det}(A') = \text{det}(A)$

二、矩阵的概念与运算

  • 矩阵的定义：由m行n列元素构成的矩形数表，称为m×n矩阵。
  • 特殊矩阵：
    • 零矩阵：所有元素都为零的矩阵。$O_{m \times n}$
    • 方阵：行数和列数相等的矩阵。$A_{n \times n}$
    • 对角形矩阵：主对角线上的元素不全为零，其它的元素都为0的方阵。$\text{diag}(a_1, a_2, \ldots, a_n)$
    • 单位矩阵：主对角线上的元素都是1的对角形矩阵。$I_n$
    • 上三角形矩阵：主对角线下方元素全为零、上方的元素不全为0的方阵。$U_{n \times n}$
    • 下三角形矩阵：主对角线上方的元素全为零，下方的元素不全为0的方阵。$L_{n \times n}$
  • 矩阵的运算：
    • 矩阵加法：同型矩阵对应元素相加。$A + B = C$
    • 矩阵减法：同型矩阵对应元素相减。$A - B = C$
    • 数乘矩阵：矩阵的每一元素都乘以常数k。$kA = C$
    • 矩阵乘法：左矩阵的列数等于右矩阵的行数时，才能进行乘法运算，结果矩阵的元素为左矩阵行元素与右矩阵列元素的对应乘积之和。$AB = C$

三、矩阵的初等变换与初等矩阵

  • 矩阵的初等变换：
    • 行初等变换：交换两行、用非0常数乘以某一行、某一行乘以常数k后加到另一行上去。$r_i \leftrightarrow r_j, \quad kr_i, \quad r_i + kr_j$
    • 列初等变换：交换两列、用非0常数乘以某一列、某一列乘以常数k后加到另一列上去。$c_i \leftrightarrow c_j, \quad kc_i, \quad c_i + kc_j$
  • 初等矩阵：由单位矩阵经过一次初等变换而得到的矩阵，其性质包括左乘初等矩阵相当于对原矩阵进行相应的行初等变换，右乘初等矩阵相当于对原矩阵进行相应的列初等变换，且初等矩阵均可逆。$E_{ij}, \quad E_{i}(k), \quad E_{ij}(k)$

四、逆矩阵的概念与性质

  • 逆矩阵的定义：对于n阶方阵A，如果存在n阶方阵B，使得AB=BA=E，则称矩阵B为方阵A的逆矩阵，记作B=A^-1。$AB = BA = I_n$
  • 逆矩阵存在的充分必要条件：方阵A可逆当且仅当其行列式det(A)≠0，且A^-1=1/det(A)·A*，其中A为A的伴随矩阵。$\text{det}(A) \neq 0 \quad \Rightarrow \quad A^{-1} = \frac{1}{\text{det}(A)} A^$
  • 逆矩阵的性质：
    • 逆矩阵是唯一存在的。$\text{If } A^{-1} \text{ exists, then it is unique.}$
    • 若A可逆，则A^-1也可逆，且(A^-1)^-1=A。(A^-1)^{-1} = A
    • 若A可逆，数λ≠0，则(λA)^-1=1/λ·A^-1。$(\lambda A)^{-1} = \frac{1}{\lambda} A^{-1}$
    • 若A、B为同阶可逆矩阵，则(AB)^-1=B^-1A^-1。$(AB)^{-1} = B^{-1} A^{-1}$
    • 若A可逆，则(A^T)^-1=(A^-1)^T。$(A^T)^{-1} = (A^{-1})^T$

第二部分：行列式与矩阵的应用

一、行列式在解线性方程组中的应用Cramer法则

  • Cramer法则：对于线性方程组Ax=b，如果系数矩阵A的行列式det(A)≠0，则方程组有唯一解，解为x_i=det(A_i)/det(A)，其中A_i是将A的第i列替换为常数项向量b得到的矩阵。$x_i = \frac{\text{det}(A_i)}{\text{det}(A)}$

二、矩阵在解线性方程组中的应用

  • 线性方程组的矩阵表示：线性方程组Ax=b可表示为矩阵形式，其中A为系数矩阵，x为未知数向量，b为常数项向量。$Ax = b$
  • 利用矩阵的初等变换解线性方程组：通过对方程组的增广矩阵进行初等行变换，将其化为行最简形，从而求得方程组的解。$\text{rref}([A|b]) = [I_n|x]$

第四部分：矩阵的秩与分块矩阵

一、矩阵的秩

  • 矩阵秩的定义：矩阵的秩是指矩阵中行向量或列向量的最大线性无关组所含向量的个数。
  • 求矩阵秩的方法：
    • 利用初等变换将矩阵化为行阶梯形矩阵，非零行（或列）的个数即为矩阵的秩。
    • 利用行列式的性质，计算矩阵的子式，非零子式的最高阶数即为矩阵的秩。

二、分块矩阵

  • 分块矩阵的概念：将矩阵划分为若干个子矩阵块，每个子矩阵块可以是方阵或非方阵。
  • 分块矩阵的运算：
    • 分块矩阵的加法：对应子矩阵块相加。
    • 分块矩阵的乘法：满足矩阵乘法的条件，即左矩阵的列数等于右矩阵的行数，然后对应子矩阵块进行乘法运算。
    • 分块矩阵的转置：将分块矩阵的子矩阵块进行转置，并调整子矩阵块的位置。

第五部分：矩阵的特征值与特征向量

一、特征值与特征向量的概念

  • 特征值的定义：对于n阶方阵A，如果存在非零向量x和标量λ，使得$Ax=λx$，则称λ为矩阵A的特征值，x为对应于特征值λ的特征向量。
  • 特征多项式：矩阵A的特征多项式为$|A-λE|$，其中E为单位矩阵，特征值即为特征多项式的根。

二、特征值与特征向量的计算

  • 计算特征值：求解特征多项式$|A-λE|=0$，得到矩阵A的特征值。
  • 计算特征向量：对于每个特征值λ，求解方程组$(A-λE)x=0$，得到对应于特征值λ的特征向量。

第六部分：矩阵的相似与对角化

一、矩阵的相似

  • 相似矩阵的定义：如果存在可逆矩阵P，使得矩阵A和矩阵B满足A=P^-1BP，则称矩阵A和矩阵B相似。
  • 相似矩阵的性质：相似矩阵具有相同的特征值，但特征向量不同。

二、矩阵的对角化

  • 对角化的条件：如果矩阵A有n个线性无关的特征向量，则矩阵A可以对角化。
  • 对角化的方法：将矩阵A的特征向量构成矩阵P，使得P^-1AP为对角矩阵，对角线上的元素为矩阵A的特征值。

第七部分：二次型与正定矩阵

一、二次型的概念

  • 二次型的定义：二次型是一个二次齐次多项式，可以表示为x^T Ax，其中A为对称矩阵，x为向量。
  • 二次型的标准形：通过变量替换，将二次型化为标准形，即平方和的形式。

二、正定矩阵

  • 正定矩阵的定义：对于对称矩阵A，如果对于任意非零向量x，都有x^T Ax>0，则称矩阵A为正定矩阵。
  • 正定矩阵的性质：正定矩阵的特征值均为正数，且其行列式大于零。

一、矩阵在实际问题中的应用

  • 线性方程组的应用：在经济学、物理学等领域，线性方程组可以用来描述多个变量之间的关系，求解线性方程组可以得到这些变量的具体值。
